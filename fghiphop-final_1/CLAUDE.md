# CLAUDE.md — 佛光嘻哈研究社網站

## 專案概述
單頁靜態網站,介紹佛光嘻哈研究社。部署在 GitHub Pages。
維護者是不會寫程式的學生,所有回覆用中文,解釋要白話。

## 設計方向(已定稿,勿擅自改風格)
- 紫黑配色 + 螢光紫(#b26cff)+ 桃紅(#ff4fa3)
- 「復古電腦多視窗」版式:CD 播放器是 .EXE 視窗、活動是便利貼、留言區是黑底終端機、照片是拍立得斜疊
- 塗鴉元素:歪斜標題配噴漆疊影、貼紙散落、牆面雜訊底紋
- 手機優先,多數訪客從 IG 連結進入

## 檔案結構
- `index.html` — 頁面結構,區塊順序:入場 → 標題 → CD 播放器 → 關於 → 照片 → 活動 → 留言 → CTA
- `css/style.css` — 全部樣式。顏色字體集中在 `:root`,改配色只動這裡
- `js/config.js` — **唯一給維護者改的檔案**:歌單、照片、活動、文字、Firebase 設定
- `js/main.js` — 播放器、照片牆、燈箱、貼紙邏輯
- `js/comments.js` — 留言區,Firebase 或 localStorage 測試模式
- `assets/audio/` — mp3 音檔
- `assets/photos/` — 照片(需壓縮,單張建議 < 500KB)

## 硬約束
- 不用任何框架、不用打包工具,保持純 HTML/CSS/JS 直接可開
- 內容變動一律進 `js/config.js`,不寫死在 HTML 裡
- 音樂不能未經互動自動播放(瀏覽器政策),入場畫面的點擊就是解法,不要移除入場畫面
- 留言的 XSS 跳脫(comments.js 的 esc 函式)不可移除
- Firebase 金鑰以外的密鑰不進 repo;Firestore 安全規則見「維護說明.md」

## 驗證方式
改完後在本機直接用瀏覽器開 index.html 檢查:
1. 入場點擊後音樂播放、CD 轉動
2. 切歌、進度條、歌單正常
3. 照片點開燈箱正常
4. 留言送出與顯示正常
5. 手機寬度(375px)排版不破版
